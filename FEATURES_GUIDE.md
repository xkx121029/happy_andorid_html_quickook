# HTML Quick View 功能使用指南

## 📚 个人离线知识库功能概览

### 1. 文件夹和标签管理

#### 创建文件夹
1. 点击右上角 **⋮** 菜单
2. 选择 **"文件夹管理"**
3. 点击 **+** 按钮创建新文件夹
4. 输入文件夹名称，点击创建

#### 创建标签
1. 点击右上角 **⋮** 菜单
2. 选择 **"标签管理"**
3. 点击 **+** 按钮
4. 输入标签名称，选择颜色，点击创建

#### 为文件添加标签
1. 点击文件右侧的 **⋮** 菜单
2. 选择 **"编辑标签"**
3. 点击可用标签进行添加
4. 也可以点击 **+** 创建新标签

#### 移动文件到文件夹
1. 点击文件右侧的 **⋮** 菜单
2. 选择 **"移动到文件夹"**
3. 选择目标文件夹

---

## 🌐 2. 离线网页保存

### 保存网页步骤
1. 打开要保存的HTML文件
2. 点击顶部菜单的 **"保存网页"** 按钮
3. 填写原始URL（可选）
4. 点击 **"保存"**
5. 等待下载完成

### 保存内容
- ✅ HTML主文件
- ✅ CSS样式文件
- ✅ 图片资源（JPG, PNG, GIF, SVG, WebP）
- ✅ JavaScript文件
- ✅ 字体文件（woff2, ttf, otf等）
- ✅ 背景图片

### 保存位置
- 所有离线资源保存在：`/data/data/com.example.htmlquickview/files/web_archives/`

### 查看离线存档
1. 进入文件详情
2. 点击 **"查看离线存档"** 查看下载的资源
3. 显示资源数量和总大小

---

## 📝 3. 批注功能

### 添加批注类型

#### 🌟 高亮
- 用于标记重要内容
- 支持多种颜色选择
- 自动保存位置信息

#### 📝 笔记
- 添加详细笔记
- 可关联到特定文本位置
- 支持多种颜色

#### 🔖 书签
- 快速定位重要位置
- 自定义书签名称
- 便于导航

### 使用批注
1. 打开HTML文件
2. 点击 **"批注"** 按钮打开批注面板
3. 选择批注类型（高亮/笔记/书签）
4. 输入内容
5. 选择颜色（可选）
6. 点击添加

### 查看和管理批注
- 在批注面板查看所有批注
- 点击批注跳转到对应位置
- 可删除不需要的批注
- 支持导出批注为HTML

### 导出批注
1. 打开批注面板
2. 点击 **"导出"** 按钮
3. 批注将保存为HTML文件
4. 可在任何浏览器中查看

## 🔍 4. 全文搜索

### 搜索功能
- 不仅搜索文件名
- 搜索HTML正文内容
- 支持标题搜索
- 智能摘要提取

### 使用搜索
1. 在首页搜索栏输入关键词
2. 搜索结果实时显示
3. 匹配的内容高亮显示
4. 显示搜索摘要（上下文）

### 搜索优化
- 首次搜索会自动建立索引
- 后续搜索速度更快
- 支持中英文搜索

### 重建索引
如果搜索结果不准确，可以重建索引：
1. 进入 **设置**
2. 点击 **"重建搜索索引"**
3. 等待索引完成

---

## 📄 5. 导出功能

### 导出格式

#### PDF导出
- 转换为PDF文档
- 适合阅读和分享
- 保持排版格式

#### EPUB导出
- 转换为电子书格式
- 支持电子书阅读器
- 自动生成目录

#### HTML导出
- 保持原始HTML格式
- 适合进一步编辑
- 无格式转换

### 使用导出
1. 打开要导出的文件
2. 点击 **"导出"** 按钮
3. 选择导出格式
4. 等待导出完成
5. 文件保存到导出目录

### 导出目录
- 所有导出文件保存在：`/storage/emulated/0/Android/data/com.example.htmlquickview/files/exports/`

### 管理导出文件
1. 进入 **"已导出文件"** 页面
2. 查看所有导出文件
3. 支持打开、分享、删除操作

## 🔧 故障排除

### 离线保存失败
- 检查网络连接
- 确认资源URL可访问
- 清理缓存重试

### 搜索无结果
- 等待索引建立完成
- 尝试重建索引
- 检查搜索关键词

### 导出失败
- 检查存储空间
- 确认文件可读
- 查看错误日志

### 应用崩溃
- 清理应用缓存
- 重启应用
- 必要时卸载重装
